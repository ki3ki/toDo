📌 ToDo List – React Application
📖 Overview

This is a simple ToDo List application built using React.js to understand the fundamentals of:

Functional Components

React Hooks (useState)

State management

Event handling

Conditional rendering

List rendering using .map()

The project was built as a hands-on exercise to understand how React manages UI updates through state changes.create-react-app/docs/troubleshooting#npm-run-build-fails-to-minify](https://facebook.github.io/create-react-app/docs/troubleshooting#npm-run-build-fails-to-minify)

🚀 Features

➕ Add new tasks

☑️ Mark tasks as selected

✅ Mark selected tasks as completed

❌ Delete tasks

🔄 Dynamic UI updates using React state

🎯 Unique task IDs using Date.now()

🏗️ Tech Stack

React.js (Functional Components)

JavaScript (ES6+)

CSS

Font Awesome Ico
